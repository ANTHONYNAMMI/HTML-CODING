CSS(Cascading Style Sheet)

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation and layout of a document written in HTML,allowing you to apply styles such as colors, fonts, spacing, and positioning.
Here css having a syntax that is SELECTOR and DECLORATION
here the selector is defined as applied to a particular HTML elements
here decloration is defined as PROPERTY which combination of PROPERTY NAME and PROPERTY VALUE which are within the flower brackets and each property is seperated by a semicolon.
FOR EXAMPLE
p{color:red;}
from this example P is selector and COLOR is a property name and RED is property value

TYPES OF CSS

INLINE CSS
INTERNAL CSS
EXTERNAL CSS

INLINE CSS

Inline css is defined as only for a particular tag
In inline css we can use style attribute

INTERNAL CSS

Internal css is defined as only apply the style for entire the webpage
In this css we can use style tag this is appicable for the webpsge

EXTERNAL CSS

External css is defined as styles for a several webpages
In this css we can create a seperate document for this css with an extension (.css)
In this css tag are not required
AND this external css is linked with the HTML document we can use the link tag with some attributes are RELATION(rel) in this attribute we can use STYLESHEET
and another attribute is HREF in this attribute we can use the EXTERNAL CSS DOCUMENT.

NOTE

LINK TAG MUST BE USED IN HEAD TAG only

DOMINANT ORDER IN DIFFERENT CSS

Inline css > Internal css > External css (if you use style tag below the link tag) > BROWSER(default styles)
Inline css > External css > Internal css (if you use link tag below the style tag) > BROWSER(default styles)
